# TarotGo Sound Files

This folder should contain the following audio files. Please add them in MP3, WAV, or M4A format.

## Required Sound Files

1. **card_flip.mp3** (or .wav/.m4a)
   - Sound when a tarot card is flipped/revealed
   - Recommended: Smooth, mystical card flip sound
   - Duration: 0.5-1 second

2. **card_shuffle.mp3**
   - Sound of cards being shuffled
   - Recommended: Gentle shuffling sound
   - Duration: 1-2 seconds

3. **card_draw.mp3**
   - Sound when selecting/drawing a card
   - Recommended: Quick, soft whoosh or draw sound
   - Duration: 0.3-0.5 seconds

4. **bell_chime.mp3**
   - Mystical bell or chime sound
   - Used for: App launch, important moments
   - Recommended: Crystal or Tibetan bell sound
   - Duration: 2-4 seconds

5. **success.mp3**
   - Success/completion sound
   - Used for: Completing readings, confirmations
   - Recommended: Positive, uplifting chime
   - Duration: 0.5-1 second

6. **whoosh.mp3**
   - Gentle whoosh/transition sound
   - Used for: UI transitions, selections
   - Recommended: Soft wind or transition sound
   - Duration: 0.3-0.7 seconds

7. **ambient_mystic.mp3**
   - Ambient background music (OPTIONAL)
   - Used for: Background during readings
   - Recommended: Calm, mystical, loopable ambient music
   - Duration: 30-60 seconds (loopable)

## Notes

- All sounds will work if they don't exist (app won't crash)
- Sounds are optional but greatly enhance the user experience
- Keep file sizes reasonable (under 500KB per file)
- MP3 format recommended for smaller file sizes
- Make sure sounds are royalty-free or properly licensed
